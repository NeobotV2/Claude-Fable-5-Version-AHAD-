# Referenzkunden-Logos

Echte Kundenlogos hier ablegen. Das Referenz-Band (`src/components/LogoMarquee.tsx`)
zeigt automatisch das Logo, sobald in `src/lib/site.ts` bei der jeweiligen
`CLIENT_REFERENCES`-Zeile `logo: '/images/clients/<datei>'` gesetzt ist.
Ohne Logo bleibt die Text-Wortmarke stehen (kein „kaputtes" Band).

## ⚠️ Rechtlicher Hinweis
Nur Logos von Kunden verwenden, die der **öffentlichen Nennung zugestimmt**
haben. Besonders prüfen: Allianz, Bundesagentur für Arbeit (Behörde).

## Format
- Bevorzugt **SVG** (scharf, klein). Sonst **PNG mit transparentem Hintergrund**.
- Möglichst einfarbig/dunkel; das Band stellt sie einheitlich in Graustufen dar
  (bei Hover farbig).
